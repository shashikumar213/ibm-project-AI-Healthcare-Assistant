AI Healthcare Assistant

An AI-powered Healthcare Assistant that predicts possible diseases from user symptoms using **Machine Learning**, **Natural Language Processing (NLP)**, and **Google Gemini AI**.

---

 Project Overview

This project allows users to enter their symptoms in natural language or manually select symptoms. The application extracts symptoms using NLP, predicts the top possible diseases using a Machine Learning model, and generates an AI-powered explanation with precautions.

---

 Features

 Natural Language Processing (NLP)

 Symptom Detection

Disease Prediction

 Top 3 Disease Prediction

 Confidence Score

 AI Explanation using Gemini AI

 Age Detection

 Duration Detection

 Manual Symptom Selection

 User Friendly Interface
